Enemy Spawner V2.0.0

Author: IamLupo

Allows you to spawn and despawn enemies. 
Set keyboard shortcuts in the mod options menu to select a breed and spawn it, as well as to despawn all enemies in the level.
Works only if you're hosting. This mod is for science only.
